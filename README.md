# proxy-server-architecture
Client–Proxy–Server architecture with a centralized proxy layer and TCP/UDP communication.

## Features
- HTTP Web Server
- Proxy Forwarding
- Proxy Caching (HIT/MISS)
- UDP QoS Measurement
- Multithreaded Connections
- TCP & UDP Communication

## Architecture
Client → Proxy → Web Server

## Ports
Proxy TCP : 8080
Server TCP : 8000
Server UDP : 9000
